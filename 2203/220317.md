[연출 사운드 시점]
문제-방법)

1. 사운드 시점과 연출시점이 다르다.
2. 사운드 노출시점확인


방법1)

애니메이션 이벤트 키로 사운드 등록  
=>유니티 애니메이션에서 프레임으로 사운드 재생 할 수 있도록 변경 

해결방법

1. 컴포넌트화시키면 사운드 재생이 안되는 경우가 있어 하드코딩.

1-1. play만 이벤트에서하고 스크립트에선 idx 플레이

=> 컴포넌트는 재생이 안되는 문제 발생이 일어날 수 있다고 함. 이에 따라 이벤트만 따로 등록하도록 분리 작업 

사실, 애니메이션 - 사운드 간의 조정 작업이었으나(클라 수정은 필요하지 않았던 것임!) but, 덕분에 나도 연출 관련 코드 개선함

수정 내용

1) 연출 전, 필요한 게임 세팅을 애니메이션 직전으로 세팅 시점을 옮김
2) 사운드 add/delete 시점도 수정

=> 렉 상황에서도 좀 더 자연스러운 연출과 사운드를 보일 수 있음..



문제)

성공 시, 연출 후 결과 화면에서 특정 사운드 소리가 재생됨

해결 방법)

유니티 프로파일러 => 애니메이션을 통해 문제가 되는 사운드를 확인함. 
단독 사운드 판단을 통해 해제 시점을 해당 사운드가 재생되어도 재생 제외 처리함
